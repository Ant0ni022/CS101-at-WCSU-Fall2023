The following demonstrate the various uses of character entities in working with non-English languages on Web pages.
Gibt es ein Café: in der Nähe?
Is there a café nearby?

Je voudrais un dîner.
I want to eat dinner.

Y una mesa por mañana, por favor.
And a table for tomorrow, please.

Oh! C'è una specialità locale?
Oh! Is there a local speciality?
